Multiplayer Poker Game

Language: Java, PHP, SQL

Group Project

Database: MySQL

This project is a project that I created to fulfill the final assignment for the Oriented Programming course Objects in semester 4 Data. This project is a turnbase multiplayer game where players are linked together a database. Game and player data is saved every time there is a change (number of chips, value and card symbols, etc.). The game will be finished if one of the players folds or beats the player in the last turn (Showdown), where the combination of card values for each player will be calculated and compared
